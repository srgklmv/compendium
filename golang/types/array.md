# Array

**Array** - это массив объектов одного типа с фиксированной длинной. Соответственно,
имеет две характеристики - длину и тип данных.

```go
type Array struct {
    len  int64
    elem Type
}
```

Представляет собой **непрерывную область памяти**.

### Декларирование

```go
var a [5]int
a := [5]int{1, 2, 3, 4, 5}
a := [...]int{1, 2, 3, 4, 5}
```
